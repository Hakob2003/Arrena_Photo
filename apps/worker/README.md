# Background Worker

Эта директория зарезервирована для воркера (отдельного Node.js процесса), который будет отвечать за фоновые задачи (Background Jobs) с использованием `BullMQ`.

В продакшен среде обработка тяжелых задач (например, ожидание генерации изображений от внешних API) должна происходить в отдельном сервисе, чтобы не блокировать основной API Gateway.

### Планируемые задачи:
- Обработка очередей генераций (Image Generation)
- Рассылка email
- Очистка устаревших файлов
- Отложенные биллинг транзакции
