# Отчет по фазе 1 (Безопасность) - Задача SEC-008

## Выполненная задача:

- **SEC-008:** Скрипт миграции старых паролей.
  - Написан скрипт `scripts/rehash-passwords.ts` для сканирования пользователей и выявления слабых хэшей (или неподдерживаемых форматов).
  - В `auth.service.ts` добавлен механизм **On-the-Fly Rehashing**: если пользователь успешно логинится, но его хэш bcrypt имеет `cost < 10` (или другой слабый/устаревший формат), сервер автоматически перехэширует его пароль с актуальными настройками безопасности и обновляет базу данных без прерывания сессии пользователя.

Все проверки `pnpm build` и `pnpm test` пройдены успешно.
