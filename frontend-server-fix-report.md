# Отчет об исправлении Internal Server Error 500

## Проблема
Пользователь столкнулся с ошибкой `Internal Server Error` (500) при обращении к главной странице `http://localhost:3000/`. В логах Next.js была замечена ошибка:
```
apps/frontend dev:  ⨯ [Error: UNKNOWN: unknown error, open 'F:\Arrena_Photo\apps\frontend\.next\static\chunks\webpack.js'] {
apps/frontend dev:   errno: -4094,
apps/frontend dev:   code: 'UNKNOWN',
apps/frontend dev:   syscall: 'open',
...
```

## Причина
Папка `.next` (кэш сборки) во фронтенде повредилась или была частично заблокирована "зависшим" (ghost) процессом Node.js, из-за чего сервер разработки Next.js не мог получить доступ к скомпилированным чанкам webpack.js. 

## Решение
1. Завершены все процессы, зависшие на портах `3000` и `4000` (чтобы освободить файловые блокировки).
2. Выполнено полное удаление папки `apps/frontend/.next`.
3. Сервер разработки запущен заново.
4. Проверка с помощью `curl http://localhost:3000/` подтвердила, что страницы снова успешно компилируются (ответ сервера `200 OK`, сборка прошла успешно).

Всё работает корректно, проблема с кэшем Next.js устранена.
